# Changelog

## [Unreleased]

### Added
- **Interactive Dashboard**: Run `g4d` to see a menu of options.
- **Platform Detection**: Automatically detects OS (Linux, macOS, Windows, WSL), Distro, and Package Manager.
- **Dependency Management**: Checks and installs system packages defined in config.
- **Stow Management**: Symlinks config files using GNU Stow logic.
- **Machine Config**: Interactive prompts for machine-specific settings (git user, email, GPG).
- **External Deps**: Clones external repositories (plugins, themes).
- **Init Command**: Scans existing dotfiles to generate `.go4dot.yaml`.
- **Doctor Command**: Health checks for your setup.
- **UI**: Beautiful TUI with colors, spinners, and progress indicators.
